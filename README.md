## О проекте

**3D Models Catalog** — это удобный каталог-агрегатор, который собирает лучшие бесплатные 3D модели с сайта free3d.com. Проект создан для быстрого поиска и перехода к скачиванию 3D моделей.

### Особенности

- **16 категорий** 3D моделей
- **Поиск** по категориям с debounce
- **Адаптивный дизайн**
- **Анимации** при наведении и появлении
- **Уведомления** при переходе на free3d.com
- **Glassmorphism** стиль

## Технологии

- **HTML5** — семантическая разметка
- **CSS3** — Grid, Flexbox, анимации, переменные
- **JavaScript (ES6+)** — поиск, уведомления, плавная прокрутка
- **Bootstrap Icons** — векторные иконки
